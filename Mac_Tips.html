快捷键 / 功能 说明 分类
一、系统基础快捷键
⌘ + C 复制选中内容（文字/文件/图片均可） 高频
⌘ + V 粘贴复制的内容 高频
⌘ + X 剪切（仅支持文件/文字） 高频
⌘ + Z 撤销上一步操作 高频
⌘ + Shift + Z 重做（撤销的撤销） 高频
⌘ + A 全选 高频
⌘ + S 保存当前文件 高频
⌘ + F 在当前窗口内搜索 / 查找 高频
⌘ + P 打印当前文档 常用
⌘ + Q 退出当前应用程序 常用
⌘ + W 关闭当前窗口（但不退出应用） 常用
⌘ + H 隐藏当前应用窗口（App 还在运行） 常用
⌘ + M 最小化当前窗口到 Dock 常用
二、窗口与管理
⌘ + Tab 快速切换到上一个使用的应用程序 高频
⌘ + ` 在同一个应用的不同窗口间切换（如多个 Chrome 窗口） 常用
⌘ + , (逗号) 打开当前应用的设置/偏好设置 常用
Control + ↑ 打开调度中心（Mission Control），鸟瞰所有窗口和桌面 高频
Control + ← / → 切换到左边 / 右边的桌面（Spaces） 高频
Control + ↓ 显示当前应用所有窗口（App Exposé） 常用
⌘ + Space 打开聚焦搜索（Spotlight），输入文件名、应用名即可快速搜索 高频
⌘ + Option + Space 直接打开 Finder 窗口进行文件搜索 常用
⌘ + Option + D 显示 / 隐藏 Dock 栏 隐藏
⌘ + Option + Esc 强制退出应用程序（类似 Windows 的 Ctrl+Alt+Del） 急救
三、文件和 Finder 技巧
⌘ + O 打开选中文件（用默认应用打开） 高频
⌘ + I 显示文件简介（右键"显示简介"也可以） 常用
⌘ + Delete 将选中文件移至废纸篓 高频
⌘ + Shift + Delete 清空废纸篓（需确认） 常用
⌘ + Shift + A / U / D / D 快速打开Applications / Utilities / 桌面 / 文稿文件夹 常用
⌘ + L 为选中文件/文件夹创建替身（Alias，类似快捷方式） 常用
⌘ + R 在 Finder 中查看原始路径（对使用替身很有用） 隐藏
⌘ + Option + V 移动文件（相当于"剪切 + 粘贴"，而非复制） 高频
⌘ + C → ⌘ + Option + V 复制文件并在目标位置粘贴（不保留原文件） 常用
空格键 选中文件后按空格，可快速预览（图片/PDF/视频均可） 高频
⌘ + 点击 Dock 图标 打开该应用对应的 Finder 目录 隐藏
拖动文件到路径栏 将文件拖到 Finder 窗口顶部路径栏上，快速导航到该目录 隐藏
四、文字处理技巧
⌘ + B 文字加粗（Bold） 高频
⌘ + I 文字斜体（Italic） 高频
⌘ + U 文字下划线（Underline） 常用
⌘ + K 添加网页链接 常用
⌘ + [ 和 ⌘ + ] 缩进减少 / 增加（类似格式化代码） 常用
⌘ + \ 在支持的应用中清除格式（粘贴纯文本） 常用
⌘ + Shift + ↑ / ↓ 向上/下选择一整段文字 常用
Option + ← / → 按单词跳跃光标位置（比只用 ← → 更快） 高频
⌘ + ← / → 光标跳到行首 / 行尾 高频
⌘ + Delete 删除光标所在行光标之前的所有文字 隐藏
五、截图 & 录屏
⌘ + Shift + 3 截取整个屏幕（保存到桌面） 高频
⌘ + Shift + 4 截取自定义区域（鼠标拖动选择区域） 高频
⌘ + Shift + 4 + 空格 截取单个窗口（鼠标变为相机图标，点击窗口） 高频
⌘ + Shift + 5 打开截图工具栏（可录屏、设置保存位置、计时等） 高频
Control + ⌘ + Shift + 3/4 截图但不保存到文件，而是复制到剪贴板（可在其他地方 ⌘ + V 粘贴） 常用
⌘ + Shift + 5 → 选项 设置截图保存位置（桌面/文稿/剪贴板/邮件等） 设置
六、触控板手势（强烈建议开启）
三指向上滑动 打开调度中心，鸟瞰所有窗口 手势
三指向下滑动 显示当前应用所有窗口 手势
三指左右滑动 切换上一个/下一个应用（类似 ⌘ + Tab） 手势
两指捏合 打开启动台（类似手机桌面） 手势
两指轻点右键 弹出右键菜单（不需要按右键） 手势
两指向右滑动 返回上一级（在 Finder 和 Safari 中） 手势
两指长按 触发用力长按（类似 3D Touch） 手势
轻拍触控板 无需按下，只要"轻拍"即可触发点击（更省力） 设置
七、系统设置优化（值得一改）
系统设置 → 触控板 → 轻拍来点按 开启后无需用力按压，轻拍即可点击，省力又保护触控板 强烈推荐
系统设置 → 触控板 → 更多手势 → 调度中心 三指向上滑动打开调度中心，比快捷键更直观 强烈推荐
系统设置 → 辅助功能 → 指针控制 → 跟踪速度 调高指针跟踪速度，鼠标操作更跟手 推荐
系统设置 → 通用 → 关于 → 存储空间 → 管理 查看存储空间使用情况，清理无用文件 推荐
系统设置 → 通用 → AirDrop 与 Handoff 开启 AirDrop 可与其他苹果设备无线传输文件 推荐
Finder → 偏好设置 → 边栏 自定义 Finder 侧边栏显示哪些常用文件夹 个性化
系统设置 → 键盘 → 键盘快捷键 自定义或查看所有系统快捷键 进阶
系统设置 → 电池 → 低电量模式 笔记本建议插电时关闭低电量模式，性能更好 性能
系统设置 → 显示器 → 夜览 开启自动护眼模式，根据日出日落调节屏幕色温 护眼
八、效率提升隐藏技巧
⌘ + Shift + T 重新打开刚才关闭的标签页（Chrome / Safari / Finder 均支持） 隐藏
⌘ + Option + E 清空聚焦搜索的历史记录 隐藏
⌘ + Shift + . 在 Finder 中显示隐藏文件（以 . 开头的文件） 隐藏
在 App 中双指长按文本 触发生成自然语言查找（Siri 建议的相关内容） 隐藏
⌘ + Option + I 在 Safari 或 Chrome 中打开开发者工具 隐藏
拖动文件时按 Option 拖动文件时按住 Option，会自动复制而非移动 隐藏
⌘ + 点击菜单项 点击菜单栏中任何一项时按住 ⌘，可以看到完整的菜单路径 隐藏
⌘ + 音量/亮度图标 点击菜单栏音量或亮度图标时按住 ⌘，可以微调（1/4 档） 隐藏
九、实用终端命令（Terminal）
defaults write com.apple.dock tilesize -int 48 将 Dock 图标尺寸调整为 48 像素（数值可改，改后需运行 killall Dock） 终端
killall Dock 重启 Dock（修改 Dock 设置后执行以生效） 终端
chflags hidden ~/文件路径 隐藏指定文件或文件夹（ Finder 中不显示） 终端
chflags nohidden ~/文件路径 取消隐藏文件/文件夹 终端
caffeinate -u -t 3600 防止 Mac 在接下来 1 小时内进入睡眠（适合挂机） 终端
caffeinate -s 防止 Mac 在插电状态下进入睡眠（直到断电） 终端
sudo pmset -a lessbright 1 低电量时自动降低亮度（需输入管理员密码） 终端
rm -rf ~/.Trash/* 清空废纸篓（慎用，会彻底删除） 终端
diskutil list 查看所有挂载的磁盘和分区 终端
top 查看当前系统资源占用（CPU/内存），按 Q 退出 终端
curl ifconfig.me 查看本机公网 IP 地址 终端
mdls ~/文件名 查看文件的所有元数据（创建时间、修改时间等） 终端
